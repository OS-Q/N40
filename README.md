## [平台描述](https://github.com/OS-Q/W24/wiki) 

数据缓存平台，作为最边缘端嵌入式CDN节点完成边缘侧的数据内容分发，提高这个低功耗网络的数据可获得性。

### [资源](OS-Q/)

- 和[DB-Q](https://github.com/OS-Q/DB-Q)同级平台的适配部署
- 和[MQ-Q](https://github.com/OS-Q/MQ-Q)同级平台的适配部署
